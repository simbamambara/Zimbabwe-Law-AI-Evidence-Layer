# Public-Minimal Data

I created the public-minimal derivative to reduce exposure while preserving enough structure for technical demonstration and review.

I exclude case names, document names, parties, counsel, factual narratives, procedural history, final orders, full judgment text, source blocks, and source paths from this layer.

This minimisation step does not, by itself, establish that publication is lawful or appropriate. Before I release any file from this directory, I require separate legal, privacy, editorial, and institutional approval.
