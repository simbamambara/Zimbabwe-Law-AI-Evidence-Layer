# Evidence metadata

This folder contains validation reports, manifests, review queues, normalization logs, classification anomalies, duplicate-source evidence, and integrity scans.

Expected evidence includes:

- `validation_report.json`
- `dataset_statistics.json`
- `dataset_manifest.json`
- `repository_manifest.json`
- `normalization_log.csv`
- `manual_review_queue.csv`
- `classification_anomalies.csv`
- `duplicate_source_occurrences.csv`
- `sensitive_content_scan.json`
