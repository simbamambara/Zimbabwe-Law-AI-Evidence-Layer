# Compliance Position

This repository demonstrates technical and organisational evidence: provenance, integrity, minimisation, access rules, incident planning, registers, privacy-impact assessment, human review, and release controls.

It does not prove that the organisation is registered or licensed, that a data-protection officer has been formally appointed, that every lawful basis is settled, or that full-text publication is authorised. Those are explicit sign-off items in the POTRAZ checklist.
