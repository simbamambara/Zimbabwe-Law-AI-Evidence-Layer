# Record of Processing Activities

The operational register is `docs/registers/processing_activities_register.csv`.

Minimum fields: processing name, purpose, controller, processor, DPO, data subjects, data categories, lawful basis, recipients, transfers, retention, security, source, systems, DPIA status, owner, and review date.

The register must be updated when the purpose, hosting, public-release model, processor, model-training use, or data categories change.
